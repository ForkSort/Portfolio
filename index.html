Time tracking dashboard

A dynamically rendered time-tracking dashboard

Demo Source

Singleton approach, responsive one-page app. To increase performance it loads JSON-data once, then renders user-data from memory. Taking that one step further, i used createElement() & createDocumentFragment() to make it both XSS-safe and faster (than innerHTML).

Tech stack: HTML, CSS, JS

Pure-CSS table-tennis

Table tennis animation using only CSS

Demo Source

Wrote this for the css-animation assignment at Kodehode. I leveraged box-shadows within animation-keyframes to give the illusion of 3d-movement.

Tech stack: HTML, CSS
